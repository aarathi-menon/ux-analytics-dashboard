# UX Behaviour Analytics Dashboard

A React-based analytics platform that tracks user interactions on a demo e-commerce website and visualises behavioural insights.

## Features

- User interaction tracking
- Page visit analytics
- Event logging
- Behaviour insights dashboard
- Data visualisation with charts

## Tech Stack

Frontend
- React
- React Router
- Recharts

Tools
- Vite
- GitHub

## Project Structure

src
components
pages
services
utils
data

## Development

Install dependencies

npm install

Run project

npm run dev
